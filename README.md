# 🌌 PocketBook AI Advisor Premium

An elegant, privacy-centered, and secure financial companion dashboard. PocketBook is a self-hostable personal planner combining real-time cloud cataloging with client-side zero-knowledge document ciphering. Style-paired with high-contrast display typography, fluid interactive cards, and high-speed multi-LLM workspace options.

## ✨ Key Features

*   **🔒 Zero-Knowledge Asset Ciphering**: All transaction titles, financial categories, budget limits, private tasks, and event schedules are systematically encrypted locally inside your browser using customized PIN-based salting before they ever touch Firebase Cloud nodes.
*   **🧠 Dual AI Hybrid Engines**: Toggle on-the-fly between **Google Gemini 3.5** and **Groq Llama 4 Scout** engines. Customize response temperature, stream analytical ledger trends, and run calendar audits securely.
*   **💼 Privacy-Centered Architecture**: Your custom AI keys remain completely inside your browser's isolated `localStorage` and act on requests during prompt runs inside temporary server-edge proxy cycles.
*   **📊 Premium Ledger Controls**: Log, map, and filter entries:
    *   **Transactions**: Multi-category financial bookkeeping with daily cash flow indicators.
    *   **Budgets**: Live alert trackers displaying visual spending thresholds.
    *   **Savings Goals**: Ambient progression bars representing dynamic milestones.
    *   **Calendar & Tasks**: Multi-view scheduling grids paired with secure task workflows.
